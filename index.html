CSS Selectors CheatSheet

Selector Example Description
.class .cinematographer selects all elements with class = "cinematographer"
* * selects ALL elements
#id #lastname selects all elements with id = "lastname"
element p selects all elements <p>
element.class p.cinematographer selects all elements <p> with class = "cinematographer"
elementA elementB div p selects all elements <p> inside <div>
